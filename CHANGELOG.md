# Changelog

All notable changes to Lantern are documented here.

## 1.0.0 — 2026-07-10

- Added the installable Lantern Agent Skill.
- Added a Project Kit for accounts without uploaded Skills.
- Added guided, resumable onboarding.
- Added separate-Project life architecture and ChatGPT Library snapshots.
- Added queued requests, small direct work, background work packets, and an optional Dispatcher pattern.
- Added tailored capability and app recommendations.
- Added Luna-first GPT-5.6 routing with evidence-based escalation.
- Added privacy, approval, prompt-injection, academic-integrity, career-integrity, recovery, and migration rules.
- Added deterministic packaging, checksums, privacy scanning, and scenario tests.
