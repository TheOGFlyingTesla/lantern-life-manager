# Lantern 1.0.0

Lantern 1.0 introduces a friendly, honest personal life manager for ChatGPT Work.

## Included downloads

- `lantern-life-manager-v1.0.0.zip` — installable Agent Skill for accounts with Skills.
- `lantern-project-kit-v1.0.0.zip` — no-code Project instructions for accounts without Skills.
- `SHA256SUMS` — checksums for both archives.

## Highlights

- One Life Manager Project and separate domain Projects from the beginning.
- ChatGPT Library-backed append-only snapshots with visible manual fallback.
- Guided, resumable, one-question-at-a-time onboarding.
- Queued requests, small direct work, and optional background Work packets.
- Tailored capability and app recommendations.
- Luna-first GPT-5.6 routing with selective escalation.
- Exact-action approval, privacy defaults, untrusted-content handling, recovery, and migration.

## Validation

The release source is covered by unit tests, the official Skill validator, repository checks, deterministic package tests, archive inspection, privacy scanning, and isolated forward-test scenarios.

Account-dependent Library retrieval and background behavior can vary by plan, memory mode, device, and rollout. Lantern provides explicit manual fallbacks and does not claim those paths succeeded without evidence.
