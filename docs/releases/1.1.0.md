# Lantern 1.1.0

Lantern 1.1 makes first-time setup faster, clearer, and more reliable for people using agentic AI in everyday life.

## What changed

- Framework-first onboarding establishes the complete Life Manager and justified domain structure before detailed domain work.
- When supported, a background setup task prepares the framework while the main task continues the friendly guided interview.
- Sequential and consolidated manual fallbacks prevent setup from stopping after every mechanical step.
- macOS and Windows instructions now distinguish Lantern from optional Codex or operating-system prerequisites.
- Browser guidance distinguishes the signed-in Work app from a separately authenticated controlled browser.
- Model routing now favors cost-sensitive Luna use, with Sol Medium as the normal ceiling for ambiguous or consequential work.

## Release assets

- `lantern-life-manager-v1.1.0.zip` — installable Skill package.
- `lantern-project-kit-v1.1.0.zip` — manual no-developer-tool Project Kit.
- `SHA256SUMS` — checksums for both archives.

## Validation

The release runs the unit suite, repository validator, privacy scanner, deterministic package checks, checksum verification, and Skill package validation. Account-dependent ChatGPT Project creation and background-task availability retain explicit sequential and manual fallbacks.
