# Lantern 1.1.1

Lantern 1.1.1 removes an unnecessary obstacle from first-time installation.

## Fixed

- End users are no longer asked to install GitHub Desktop, GitHub CLI, or Git.
- Installation no longer asks users to create or sign in to a GitHub account.
- Codex-assisted setup now starts with an anonymous release download and falls back to a normal browser download or the manual Project Kit.
- Repository cloning and bundled Git are reserved for explicitly requested source-development work.
- After installation, Codex now invokes Lantern automatically or gives one exact fresh-task prompt when automatic invocation is unavailable.
- Domain-neutral framework preparation begins before the first onboarding answer.

## Release assets

- `lantern-life-manager-v1.1.1.zip` — installable Skill package.
- `lantern-project-kit-v1.1.1.zip` — manual no-developer-tool Project Kit.
- `SHA256SUMS` — checksums for both archives.
