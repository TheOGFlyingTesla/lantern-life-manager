# Lantern 1.2.0

Lantern 1.2 turns first-time setup into a short, visible, agentic workflow.

## What changed

- Setup now shows a four-step progress card and questions remaining on every onboarding turn.
- Initial setup asks no more than three questions; skipping advances instead of starting a loop.
- The full starter Project architecture is shown immediately, including separate Home, Finances, Appointments/Admin, career, active roles, school, wellbeing, personal, and relationship/community areas.
- When Computer Use is available, Lantern creates, configures, and verifies ChatGPT Projects instead of giving the user a list of clicks.
- When the desktop app cannot be controlled, Lantern opens ChatGPT web, confirms that the user sees the login page, asks them to sign in, verifies authentication, and politely asks permission before moving the cursor.
- If automatic creation is genuinely unavailable, Lantern says so and returns one consolidated manual setup packet.

## Release assets

- `lantern-life-manager-v1.2.0.zip` — installable Skill package.
- `lantern-project-kit-v1.2.0.zip` — manual Project Kit.
