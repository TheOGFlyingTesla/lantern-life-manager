# Lantern 2.0.0

Lantern 2 is the easy-to-install desktop release.

Download **`lantern-desktop-v2.0.0.zip`**, extract the ready-made `Lantern` folder, add it as one local Project in the ChatGPT desktop app, and type:

> Start Lantern.

The folder already contains the Life Manager, dashboard, nine starter domains, durable status and request files, and templates for adding new domains. The Life Manager is the only required task; focused coordinator tasks start on demand.

Lantern 2 requires no Skill installation, Git, GitHub account, Python, terminal, or developer tools. It intentionally does not support the web version because the web interface cannot provide the same dependable local file coordination.

The separate `lantern-skill-v2.0.0.zip` asset is optional and intended only for experienced users who specifically want an installed Codex Skill. It augments, but does not contain or replace, the desktop workspace. `SHA256SUMS` verifies both downloads.

This is a breaking architecture change from Lantern 1.x. Preserve useful 1.x information, migrate it into the new local folder, and verify the dashboard before deleting old Projects or chats.
