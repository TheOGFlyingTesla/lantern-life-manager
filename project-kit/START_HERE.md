# Start Lantern without installing a Skill

Use this kit when your ChatGPT account does not show **Profile → Skills**.

1. Create a ChatGPT Project named **Lantern — Life Manager**.
2. Open `LIFE_MANAGER_INSTRUCTIONS.md`, copy all of it, and add it to that Project's instructions.
3. Upload the four files from `templates/` to the Life Manager Project or keep them in ChatGPT Library for reuse.
4. Start a new chat inside the Project and say: **Help me set up Lantern.**
5. Answer the short minimum-domain questions. Lantern will prepare the entire justified framework before asking detailed questions about any one life area.
6. For each recommended domain Project, copy in `DOMAIN_COORDINATOR_INSTRUCTIONS.md` after replacing `[DOMAIN NAME]` and `[DOMAIN ID]`.
7. Upload `templates/domain-snapshot.md` and `templates/work-request.md` to each domain Project, or paste their contents into a starter chat there. A separate domain Project cannot see files uploaded only to the Life Manager Project.

You can stop onboarding and return later. Lantern saves a `lantern-onboarding/v1` checkpoint after each section. Say **Help me recover Lantern** to resume from the latest confirmed information.

Lantern setup does not require Git, Python, Xcode, Apple developer tools, or other developer software on macOS or Windows.
