# Start Lantern without installing a Skill

Use this kit when your ChatGPT account does not show **Profile → Skills**.

1. Create a ChatGPT Project named **Lantern — Life Manager**.
2. Open `LIFE_MANAGER_INSTRUCTIONS.md`, copy all of it, and add it to that Project's instructions.
3. Upload the three files from `templates/` to the Project or keep them in ChatGPT Library for reuse.
4. Start a new chat inside the Project and say: **Help me set up Lantern.**
5. Answer one question at a time. Lantern will recommend separate Projects for the life areas that are actually active.
6. For each recommended domain Project, copy in `DOMAIN_COORDINATOR_INSTRUCTIONS.md` after replacing `[DOMAIN NAME]` and `[DOMAIN ID]`.

You can stop onboarding and return later. Say **Help me recover Lantern** to resume from the latest confirmed information.
