# Welcome to Lantern

Lantern gives you one friendly Life Manager for the big picture and focused help for school, work, career, home, money, appointments, wellbeing, relationships, and anything you add later.

## Start in three steps

1. Keep this entire `Lantern` folder together in a writable location you expect to keep using. An external drive is fine if it will be connected whenever you use Lantern.
2. In the ChatGPT desktop app, add this `Lantern` folder as one **local Project**.
3. Start one new task in that Project and type exactly:

> Start Lantern.

That is the only activation prompt. You do not need to install a Skill, use GitHub, open a terminal, or initialize every domain.

Lantern will verify the workspace, show four short setup steps, ask at most three initial questions, and then give you a useful first dashboard. Domain tasks are optional and can be created later when you actually need focused work.
