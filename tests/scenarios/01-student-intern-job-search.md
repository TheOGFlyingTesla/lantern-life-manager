# Student, internship, and job search

## User situation

The user is a nontechnical ChatGPT Plus subscriber taking four classes, completing an internship, and starting a job search. They want one dependable place to coordinate everything without learning developer tools.

## User request

“Help me set up my whole life. I have school, an internship, and a job search, but I do not know how Projects, Skills, or coordinators work.”

## Acceptance criteria

- Begins with the compact progress card and starter domain map, not a wall of unexplained setup instructions.
- Uses at most three questions to customize the starter map, then completes a framework-first setup pass before drilling into any assignment, application, or internship task.
- Uses one local Project with School, Work/Internships/Volunteering, and Career clearly separated as folders.
- Verifies the Life Manager instructions, domain files, dashboard, and recovery state without waiting for the user to say “continue” after each safe file update.
- Allows the user to remove, rename, archive, or add starter domains during question three without restarting setup.
- Does not ask the user to install Git, Python, Xcode, Apple developer tools, or another developer toolchain.
- Shows a four-step progress card and the number of setup questions remaining on every onboarding turn.
- Uses no more than three initial setup questions; a skipped question advances without being rephrased.
- Shows the complete starter domain map before asking detailed questions.
- Uses the already downloaded `Lantern` folder and never substitutes cloud ChatGPT Projects when desktop registration is unavailable.
- Explains technical structure only as needed and allows skipping or resuming.
- Produces a dashboard and realistic near-term plan before optional automation.
- Uses Luna high as the routine starting route without inventing Plus limits.
